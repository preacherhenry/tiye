Usage: null resolve [options] [searchPaths...]

Options:
  -e, --exclude <exclude...>    Package names to exclude when looking up for
                                modules.
  -p, --platform [platform]     The platform that the resulting modules must
                                support. Available options: "apple", "android"
                                (default: "apple")
  --project-root <projectRoot>  The path to the root of the project. Defaults
                                to current working directory (default:
                                "C:\\Users\\lenovo\\taxi_mobile\\android")
  -j, --json                    Output results in the plain JSON format.
                                (default: false)
  -h, --help                    display help for command
